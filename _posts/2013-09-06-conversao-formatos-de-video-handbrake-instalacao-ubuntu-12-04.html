Para se poder efectuar a conversão entre formatos de vídeo poderemos usar o handbrake.

Para instalar no ubuntu 12.04 de forma rápida:

Abrir uma janela de terminal e digitar os seguintes comandos para adicionar o PPA à sources.list

Adiciona o repositório PPA 
sudo add-apt-repository ppa:stebbins/handbrake-releases

Actualizar a lista dos pacotes disponíveis 
sudo apt-get update

Instalar o handbrake 
sudo apt-get install handbrake-gtk handbrake-cli


Nota: usem de forma responsável e nunca infringindo a legislação vigente e direitos de autor aplicavéis

Nota 2: Não experimentei em versões mais recentes do ubuntu, mas as instruções aqui mencionadas também deverão funcionar.
