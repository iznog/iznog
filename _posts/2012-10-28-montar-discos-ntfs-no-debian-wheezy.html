--- layout: post title: Montar discos NTFS no debian wheezy date: 2012-10-28 comments: false ---
Ao utilizar discos externos formatados em NTFS no meu debian wheezy, os mesmos eram sempre montados como leitura.

Para montar os mesmos como leitura/escrita só é preciso instalar o ntfs-3g que facilmente se consegue executando o seguinte comando numa linha de comandos:

sudo apt-get install ntfs-3g
