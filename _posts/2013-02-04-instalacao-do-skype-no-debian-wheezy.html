Para instalar o skype no debian wheezy, deveremos aceder ao site www.skype.com e descarregar a versão para debian 7.0 (multiarch).

Abrir uma consola

Verificar qual a arquitectura que está instalada (caso não saiba):
uname -a
Caso a instalação seja de 32 bits então deverá aparecer no resultado a referência a "i686", no caso de 64 bits, deverá aparecer "amd64"
No caso de instalação na arquitectura de 32 bits, deverá ser necessário apenas efectuar os seguintes comandos: 
su
dpkg -i caminho_para_o_ficheiro_instalacao_skype
apt-get -f install
No caso de instalação na arquitectura de 64 bits, deverá ser necessário verificar quais as arquitecturas para o qual o dpkg está configurado para instalar: 
su
dpkg --print-foreign-architectures
Caso não apareça como resultado "i386", deveremos efectuar os seguintes comandos: 
dpkg --add-architecture i386
apt-get update
De seguida efectuar os comandos seguintes para efectuar a instalação do skype: 
dpkg -i caminho_para_o_ficheiro_instalacao_skype
apt-get -f install
Estes passos deverão ser os necessários para proceder à instalação da actual versão do skype para debian 7.0 já suportando o "multiarch"
