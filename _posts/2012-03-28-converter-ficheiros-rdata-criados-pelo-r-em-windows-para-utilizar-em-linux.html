--- layout: post title: Converter ficheiros RData criados pelo R em Windows para utilizar em Linux date: 2012-03-28 comments: false ---
Começando pelo inicio, o R é uma aplicação de calculo estatistico, opensource.
Sendo esta aplicação usada numa cadeira de Estatistica que frequento, surgiu um problema com alguns ficheiros do R assim como ficheiros de texto, cujo formato não estaria adequado para usar no R em Linux.
O problema no caso, seria que os encode dos ficheiros disponibilizados estariam no formato ANSI em vez de UTF8 (por exemplo).

Assim para os ficheiros de texto basta fazer o seguinte comando:

iconv -c --from-code ANSI --to-code UTF8 mundo.txt >mundo_utf8.txt

Para se poder usar os ficheiros de dados do R criados em Windows teremos que recorrer a uma instalação do R em Windows e exportar os dados, executar o comando acima ao ficheiro com os dados exportados (obviamente adaptando para o nome de ficheiro adequado) e importar o ficheiro convertido, para o R em Linux.

Caso alguém conheça uma forma melhor, agradeço que comente de forma a poder melhorar o artigo.
