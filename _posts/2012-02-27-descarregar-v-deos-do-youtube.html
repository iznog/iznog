Para guardar localmente um vídeo do youtube para visualizar mais tarde offline, existem várias soluções, mas aquela que uso com mais regularidade, é o youtube-dl.
Como uso Linux, normalmente uma variante do ubuntu ou o próprio (ou pelo menos é o que tenho instalado e vou usar como referência o xubuntu 11.04 ), para instalar basta fazer:
sudo apt-get install youtube-dl
Para ver quais as opções e a forma de utilizar podemos fazer 
man youtube-dl.
Mas rapidamente basta usar o comando youtube-dl seguido do url do vídeo.
Uma nota para os mais distraídos... o URL do vídeo é o endereço que aparece na barra de endereços do browser.
Ao efectuar download se obtiverem um erro "ERROR: no fmt_url_map or conn information found in video info", façam uma actualização do youtube-dl com o comando:
sudo youtube-dl --update
Esta última obtive na discussão sobre o erro no github para o projecto youtube-dl aqui.
