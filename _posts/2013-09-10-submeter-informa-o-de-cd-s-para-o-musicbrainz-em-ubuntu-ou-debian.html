--- layout: post title: Submeter informação de CD's para o MusicBrainz em Ubuntu (ou debian) date: 2013-09-10 comments: false ---
Ao coloca um CD de audio no leitor, muitos serão os utilizadores que irão obter informação sobre o CD, suas faixas, duração, imagens dos CD's, etc, a partir de uma base de dados como o MusicBrainz.

Mas para podermos usar a informação constante nesta, alguém também terá que submeter os dados, visto que informação é obtida através da sua comunidade de utilizadores.

Assim se pretender submeter ou actualizar informação de um CD (como por exemplo o disc ID) para o MusicBrainz, poderá usar a aplicação MusicBrainZ Picard.


Para instalar no Ubuntu (testado na 12.04) poderá usar a versão que está nos repositório ou alternativamente  a versão disponibilizada através do PPA (Personal Package Archives) da MusicBrainz:

Para a segunda alternativa deve-se proceder da seguinte forma (garantirá a versão mais recente):

Abrir um terminal e depois executar os seguintes comandos: 
sudo apt-add-repository ppa:musicbrainz-developers/stable

Confirmar com a tecla ENTER após o surgimento da questão se pretende adicionar o repositorio
Actualizar a lista de pacotes com 
sudo apt-get update

Instalar a aplicação "Picard" 
sudo apt-get install picard

Se tudo correr normalmente deverá estar agora disponível a aplicação Picard na sua versão mais recente.
