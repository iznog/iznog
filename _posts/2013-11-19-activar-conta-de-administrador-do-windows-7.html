--- layout: post title: Activar conta de administrador do Windows 7 date: 2013-11-19 comments: false ---
Para activar a conta de administrador do Windows 7, deverá tomar os seguintes passos:

  • Iniciar uma consola de linha de comandos com privilégios de administrador.
  • Executar o seguinte comando:

    • net user administrator /active:yes

Para desactivar, substituir o 
/active:yes
por 
/active:no

Deverá criar uma palavra passe para essa conta, por razões de segurança.
