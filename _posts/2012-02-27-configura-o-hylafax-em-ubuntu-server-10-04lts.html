Após a configuração do hylafax no ubuntu server 10.04LTS, poderá acontecer que o ficheiro seja sempre enviado em branco.
Para corrigir este problema deve-se editar o ficheiro /var/spool/hylafax/bin/pdf2fax.gs e acrescentar à string do comando para conversão o seguinte parametro -dMaxStripSize=0 \
exemplo:
$PS -q \
    -sDEVICE=$device \
    -dNOPAUSE \
    -dSAFER=true \
    -dMaxStripSize=0 \
    -sPAPERSIZE=$paper \
    $FIXEDWIDTH \
    -dBATCH \
    -r$hres\x$vres \
    "-sOutputFile=$out" \
    $fil
