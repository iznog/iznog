Tenho por norma um repositório subversion para guardar alguns conteúdos que normalmente uso e eventualmente terei a necessidade de reaver versões anteriores.

No entanto é necessário também fazer backups.

Para efectuar backup do repositorio executo o comando:
svnadmin dump /caminho_do_repositorio >backup_repositorio.svn
gzip backup_repositorio.svn

Para efectuar a reposição do mesmo poderemos fazer:
gunzip backup_repositorio.svn.gz
svnadmin create /caminho_novo_repositorio
svnadmin load /caminho_novo_repositorio <backup_repositorio.svn
