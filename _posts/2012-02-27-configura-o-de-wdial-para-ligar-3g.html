Para se poder ligar através de modem 3G pode-se usar as configurações como se segue:
[Dialer tmn]
Modem = /dev/ttyUSB0
Stupid Mode = 1
Dial Command = ATDT
Phone = *99#
Baud = 460800
ISDN = 0
Username = tmn
Password = tmn
Modem Type = Analog Modem
Init2 = ATZ
#Init3 = AT+CPIN=9862
Init4 = AT+CGDCONT=1,"IP","internet"
ISDN = 0

Dependendo dos modems usados podemos ter que mudar as configurações e por exemplo usar outra porta para os dados em vez da ttyUSB0, como por exemplo:
[Dialer Defaults]
Modem = /dev/ttyUSB2
Modem Type = Analog Modem
ISDN = 0
Baud = 460800
Dial Attempts = 1
Username = tmn
Password = tmn
Init1 = ATZ
Init2 = AT&F &D2 &C1
Init3 = ATS7=60 S30=0 S0=0
Init4 = AT+CGDCONT=1,"IP","internet"
Phone = *99***1#
Stupid Mode = 1
Estas configurações no momento já não mantenho referência exacta de quais os modems com que usei, mas do 2º exemplo julgo ter sido usado com o ZTE MF636.
