Para obter os dados constantes num campo do formulário (por exemplo o formulário das transferência de inventário), usa-se:

SELECT $[$24.U_XX_NEnc.0];

Ao activar a opção "informação de sistema" no menu "Visão", conseguimos ver alguns dos dados que necessitamos a formula acima.

No caso de uma grid:
Então $24, é o valor obtido em "Item", o "U_XX_NEnc" é a coluna.

No caso de o campo não pertencer a uma grid, o valor "U_XX_NEnc" será obtido do valor de "variable".

Assim poderemos usar por exemplo para numa pesquisa formatada obter dados de tabelas não acessiveis directamente no formulario, baseado/filtrado nos dados obtidos pela fórmula indicada.
